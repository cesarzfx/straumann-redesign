## Assignment Objectives

This project demonstrates:

### Task 1
- Removal of non-functional white space
- Implementation of a custom favicon

### Task 2
- Conversion of static cards into animated flip cards
- 3D transforms using `perspective`, `preserve-3d`, and `rotateY()`

### Task 3
- Action buttons on the back of each card linking to relevant external content

### Task 4
- CSS-only responsive navigation system
- Hamburger menu for screens < 733px
- Horizontal navigation for larger screens

### Task 5
- Integration of social media icons with working external links

### Task 6 (Advanced CSS Feature)
- **Click-to-flip functionality** using `:checked` and sibling combinators
- **Focus Mode** using the `:has()` relational pseudo-class to dynamically dim non-hovered cards
- All features implemented without JavaScript
